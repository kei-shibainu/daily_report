### その他のコマンド(見直したい部分のみ)
+ git rm --cached [file]：作業ツリーのファイルは削除しない。
+ git diff：作業ツリーとステージの差分を比較
+ git diff --staged：ステージとリポジトリの差分を比較
+ git log -n [number] --oneline：表示数を制限し、一行でコミット履歴を表示
